# JOBFLOW - Django Job Portal

JobFlow is a Django-based Job Portal application where users can view available jobs and apply by uploading their resume.

## Features
- View available jobs
- Apply for jobs
- Upload resume
- Admin panel to manage jobs
- Application status (Pending / Accepted / Rejected)

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## How to Run the Project

1. Install Django
pip install django

2. Run migrations
python manage.py migrate

3. Start server
python manage.py runserver

4. Open in browser
http://127.0.0.1:8000/

## Admin Panel
http://127.0.0.1:8000/admin/

Admin can add jobs and manage job applications.
